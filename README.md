# The Power of AND

A static, browser-based DCI Edge practice investment calculator.

## Version 0.2 changes

- Separate DCI Edge and alternative-brand operatory counts
- Half-year expansion timing from 0.5 through 5 years, plus Never
- Simplified assumptions and removal of the additional-years-worked estimate
- Retirement illustration based on the initial investment difference and profit generated during the room-count gap
- Updated charts, timeline, and explanatory copy

## Run locally

Open `index.html` in a browser.

## Deploy

Push changes to the connected GitHub repository. Cloudflare automatically redeploys the site.
